# 更新日志

## v1.1.0 (2026-05-25)

### 新增
- **定时开关**：设置页新增「启用定时备份」开关，可随时开启/关闭自动备份
- **调度快速预设**：支持一键选择常用调度（每天02:00、每天08:00、每小时、每30分钟、每周一03:00）
- **备份后删除原文件**：每个备份源独立可选「备份后删除原文件」，上传成功后自动清理本地文件
- **二维码内嵌显示**：扫码登录二维码直接显示在 GUI 窗口中，无需依赖终端渲染

### 修复
- **上传 404 错误**：修复夸克网盘 API 上传域名从 `oss-cn-shenzhen.aliyuncs.com` 变更为 `pds.quark.cn` 导致的文件上传失败
- **Windows GBK 编码崩溃**：修复终端 Unicode 字符输出导致的应用闪退
- **登录状态持久化**：扫码登录一次后 Cookie 自动保存，下次启动免重新扫码
- **大文件上传失败**：移除不可靠的多分片上传逻辑，统一使用单分片上传（支持最大 5GB）

### 变更
- 重构上传模块：独立 `src/uploader.py`，不依赖 `quarkpan` 库的上传实现
- 重构备份进度回调：CLI 和 GUI 共用同一套回调机制

## v1.0.0 (2026-05-24)

### 初始版本
- 夸克网盘自动备份工具，支持 GUI (CustomTkinter) 和 CLI 两种模式
- 二维码扫码登录、备份源管理、实时进度显示
- MD5 增量备份、多线程并发上传、定时调度
- 备份源可视化编辑（本地路径、远程目录、排除模式）
